## Featured Projects

### [PollySoft NFC-SDK](https://github.com/pollyannaanalytics/PollySoftNfc-SDK)
*Kotlin SDK for POS NFC payment terminals*

- RSA-OAEP P2PE encryption + SHA256withRSA transaction signing
- Google Play Integrity API with per-transaction nonces (replay-attack guard)
- Isolated DI scope — zero DI graph pollution in host app
- Automated versioned releases via GitHub Actions

### [JustWoo](https://github.com/pollyannaanalytics/JustWoo-KMP) *(in progress)*
*Full-stack KMP household task app — Android · iOS · Cloud*

- KMP + DDD: shared domain models consumed by Jetpack Compose and SwiftUI
- Ktor backend with JWT auth, bcrypt, and typed REST endpoints
- PostgreSQL + Redis on AWS Lightsail — session caching via Jedis
- Dockerized multi-env deployment + GitHub Actions CI/CD
